# 月度复盘：{{date}}

> 需要安装 [Dataview](https://github.com/blacksmithgu/obsidian-dataview) 插件。未安装时，使用下方「手动填写区」。

---

## 一、本月 Plan 汇总（Dataview）

```dataview
TABLE status as "状态", file.folder as "分类"
FROM "Plans"
WHERE file.ctime >= date(this.file.name) AND file.ctime < date(this.file.name) + dur(1 month)
SORT file.ctime DESC
```

> 无 Dataview 时：手动列出本月主要 plan（路径 + 状态）。

---

## 二、本月 Contexts 汇总（Dataview）

```dataview
TABLE tags as "标签"
FROM "Contexts"
WHERE file.ctime >= date(this.file.name) AND file.ctime < date(this.file.name) + dur(1 month)
SORT file.ctime DESC
```

## 三、学习周复盘引用（Dataview）

```dataview
LIST
FROM "Contexts/LLM学习/复盘"
WHERE file.ctime >= date(this.file.name) AND file.ctime < date(this.file.name) + dur(1 month)
```
