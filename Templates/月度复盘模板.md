# 月度复盘：{{date:YYYY-MM}}

> 需要安装 [Dataview](https://github.com/blacksmithgu/obsidian-dataview) 插件。未安装时，手动填写下方表格。

## 本月 Plan 汇总（Dataview 自动）

```dataview
TABLE status as "状态", file.folder as "分类"
FROM "Plans"
WHERE file.ctime >= date({{date:YYYY-MM}} + "-01") AND file.ctime < date({{date:YYYY-MM}} + "-01") + dur(1 month)
SORT file.ctime DESC
```

## 本月 Contexts 汇总

```dataview
TABLE tags as "标签"
FROM "Contexts"
WHERE file.ctime >= date({{date:YYYY-MM}} + "-01") AND file.ctime < date({{date:YYYY-MM}} + "-01") + dur(1 month)
SORT file.ctime DESC
```

## 踩坑记录

```dataview
LIST
FROM #复盘/踩坑
WHERE file.ctime >= date({{date:YYYY-MM}} + "-01")
```
