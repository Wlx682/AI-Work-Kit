## 二、需求条目

### 2.1 首页进页会员弹窗 ✅ 已交付

**范围**：进首页后读缓存检查会员；**不改**启动 / PrepareChat / 连云链路。

| 状态 | 行为 |
|------|------|
| 有效会员 | 不弹窗 |
| 过期 | 标题「会员已过期」；正文含纳米 Work + 15 天保留 |
| 未开通 | 「开通纳米Work会员，解锁全部功能」 |
| 按钮 | 「稍后再说」/「立即开通」 |

**触发**（`NMHomeShellViewController` → `showPostConnectPopups`）：

1. Gateway 已连接直进首页  
2. `NMPrepareChatController` dismiss 后  
3. 其他 modal 挡住 PrepareChat 时仍检查  

**弹窗顺序**：会员窗 → 版本更新（`NMMainTabBarController` 不再抢先版本检查）

**实现要点**：

- `NMMembershipGuard.showExpirationPopupIfNeeded` + `expirationPopupCopy`
- `NMMembershipExpirationPopupController` 支持 `confirmTitle`
- Debug：**Mock 首页 · 会员弹窗** 验三态

**过期正文（本地写死）**：

```
续费后可继续使用 纳米Work
已为你保留历史数据 15 天
续费后可恢复使用
```

**交付 commit**：`60525365`（及 Debug `5a9dc03c`）

**背景（不开发）**：PRD §2.1 新用户加载页 / 静默创建云龙虾 — 沿用现有链路，会员失败最终由本弹窗承接。
