# 概念：大语言模型（LLM）

> **学习顺序：第 1 课起点。**

## 一句话

根据上文**预测下一个 token** 的神经网络；Chat 是它的产品形态之一。

## 关键术语

| 术语 | 含义 | 和写 prompt 的关系 |
|------|------|-------------------|
| Token | 模型读写的最小单位（非严格等于一个字） | 上下文越长，token 越多，越贵、越慢 |
| 上下文窗口 | 一次能容纳的最大 token | 超长材料要先摘要或 RAG |
| Temperature | 随机性：低=稳，高=发散 | 排查用低，头脑风暴可略高 |
| 系统提示 System | 角色与边界 | 类似 `.cursorrules` |
| 用户提示 User | 当次任务 | 类似你发的 `/template-generator ...` |

## 常见误区

- ❌ 模型「真的理解」—— 更接近统计模式匹配 + 推理链
- ❌ 上下文无限 —— 有窗口上限
- ❌ 温度越低越好 —— 创造性任务需要适当随机性
