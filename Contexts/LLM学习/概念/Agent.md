# 概念：Agent（智能体）

> **学习顺序：第 4 课。** 先会写 [[提示词工程]]，再理解 [[RAG]]。

## 一句话

能**自主规划多步**、**调用工具**、根据中间结果调整行动的 AI，而不只是单次问答。

## Agent 循环（简化）

```
目标 → 思考/规划 → 选工具 → 执行 → 观察结果 → 再思考 → … → 完成
```

## 和 Chat 的区别

| Chat | Agent |
|------|-------|
| 你问一句答一句 | 可连续多步 |
| 通常不调工具 | 可调文件、搜索、终端、MCP |
| 进度靠你带 | 可自己读 plan、更新状态 |

## Cursor Agent 在你 Kit 里的表现

- 说「续做」→ 读 plan → 输出下一步（像小型 Agent）
- 开 MCP → 自己 `obsidian_search` 再回答
- 写代码 → 读规范、搜仓库、改文件

## 相关概念

- **Tool / Function Calling**：Agent 调用的具体能力（读文件、跑命令）
- **Memory**：跨会话记忆（你的 plan + Contexts 就是人工记忆）
- **Orchestration**：多 Agent 分工（进阶，暂不必深钻）
