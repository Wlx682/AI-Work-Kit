# AI-Work-Kit 索引

> 个人 AI 协作知识库 · [[学习路线-LLM与提示词|LLM 学习]] · [[落地计划|执行清单]] · [[分享包-快速开始|5 分钟上手]]

## 一句话

**Obsidian 存模板和规范，Cursor 执行任务；链接和进行中的 plan 不长期沉淀。**

---

## 目录结构

| 路径 | 放什么 | 是否长期沉淀 |
|------|--------|--------------|
| [[Templates/]] | 排查 / 方案 / review / Figma 模板 | ✅ 是 |
| [[Contexts/Figma/]] | 公共设计规范 + 最佳实践 | ✅ 是（无链接、无功能清单） |
| [[Contexts/]] | 会议、调研、决策、踩坑 | ✅ 是 |
| [[Contexts/LLM学习/]] | LLM/提示词/Agent/Skill/MCP 概念与笔记 | ✅ 是 |
| [[Skills/]] | 提示词笔记（`@` 引用） | ✅ 是 |
| [[Plans/]] | **仅进行中的任务** | ❌ 做完可删 |
| `.cursor/skills/` | Cursor 自动 Skill | ✅ 是 |
| `.cursor/mcp.json` | enquire-mcp（已配置） | ✅ 是 |

```
Plans/
├── Bug排查/
├── 客户端技术方案/
├── 服务端技术方案/
├── 代码重构/
├── 需求分析/      ← PRD 分析，P0 闭环后可删
├── 功能开发/      ← 模块（需求+方案+界面），做完可删
├── 界面开发/      ← 仅 UI 子任务，做完可删
└── 学习/          ← 学习 plan，做完可删
```

## 高频任务速查（复制即用）

### 在 AI-Work-Kit 工作区内

| 任务 | 说法 |
|------|------|
| 排查 bug | `@Skills/template_generator.md 任务类型=排查，背景=...` |
| 续做 | `@Skills/resume_assistant.md 续做，plan文件名=Bug排查/xxx.md，当前进度=...` |
| 客户端方案 | `@Skills/template_generator.md 任务类型=客户端方案，背景=...` |
| 服务端方案 | `@Skills/template_generator.md 任务类型=服务端方案，背景=...` |
| Code Review | `@Skills/template_generator.md 任务类型=review，背景=...` |
| Figma 界面 | `@Skills/figma_ui_assistant.md` + **当次粘贴 Figma 链接** |
| 月底复盘 | `@Skills/review_assistant.md 复盘时间段=...` |
| 语义搜索 | 「知识库里有没有关于 xxx 的记录？」（需 MCP 绿灯） |
| 学习 / 续学 / 考我 | `/learn-assistant` 或 `@Skills/learn_assistant.md` |

### 在业务代码项目里（推荐）

| 任务 | 说法 |
|------|------|
| 排查 | `/template-generator 任务类型=排查，背景=...` |
| 续做 | `/resume-assistant 续做，plan=Bug排查/xxx.md，进度=...` |
| Figma | `/figma-ui-assistant 新任务，Figma=【粘贴链接】，平台=iOS，页面=...` |
| 复盘 | `/review-assistant 复盘时间段=...` |

> 全局 Skill 在 `~/.cursor/skills/`；`@` 只有打开本 Vault 时才能引用 `Skills/*.md`。

## 沉淀原则（重要）

| 沉淀到知识库 | 不沉淀 |
|--------------|--------|
| Templates、公共规范、最佳实践 | Figma 链接 |
| 会议/调研/决策（Contexts） | 已开发完的功能清单 |
| Skills 提示词 | 进行中的 plan（或做完删除） |
| 踩坑与复盘结论 | 具体 Frame / node-id |
